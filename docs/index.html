1. Introduction

This is the documentation of tenant-security microservice.

2. Tenants API

Collection of CRUD API endpoints used to manipulate tenants registered in the application.

2.1. Get All Tenants

Obtains a list of tenants registered in the application.

2.1.1. Response Body

Path Type Description

[]

Array

Tenant list

[].name

String

Name of tenant

[].displayedName

String

Displayed name of tenant

[].description

String

Description

2.1.2. Sample Request

GET /tenant HTTP/1.1
Host: localhost:8080

2.1.3. Sample Response

HTTP/1.1 200 OK
Content-Type: application/json
Content-Length: 377

[ {
  "name" : "e00df63f-aa56-4b40-8675-f0a1d0a7bfb5",
  "displayedName" : "My Project",
  "description" : "top secret"
}, {
  "name" : "110304fc-f763-458f-becc-735db4bb559d",
  "displayedName" : "My Project",
  "description" : "top secret"
}, {
  "name" : "c8aab573-3192-4a43-9223-39345aa46fda",
  "displayedName" : "My Project",
  "description" : "top secret"
} ]

2.1.4. CURL sample

$ curl 'http://localhost:8080/tenant' -i -X GET
